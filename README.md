# CrumpHome

A small set of what I'm doing to integrate my Moteino based sensor nodes into OpenHAB. There are the arduino sketches and the OpenHAB configs. My Moteino Gateway is paired with the MightyBoast shield so it can act as the RF gateway for the RPi and acts battery backup as well.
